# Shadow-Key
幽影之钥：实现隐蔽权限维持与提权  
# 🌟 工具名：夜行小偷
一个实验性红队工具，用于 Session 0 权限维持与提权实验。  
⚠️ 仅供 网络安全学习、渗透测试与教育研究 使用，禁止非法使用。  

# 🧩 功能概览
- 将自定义 DLL/Payload 注入 Session 0 服务进程  
- 代码执行上下文从 用户会话 提升至 系统服务会话
- 实现 权限维持（Persistence）
- 在特定场景下实现 权限提升（Privilege Escalation）
- 高隐蔽性，避开大部分基于用户会话的安全监控
# 💻 使用示例
- 选择目标服务进程（示例进程：svchost.exe）
- 设置要注入的 DLL 路径（绝对路径）
```test.exe 12346 C:\Temp\a.dll```
- 工具会自动将 DLL 注入目标进程并执行

# 📚 原理说明
- Session 0 注入：Windows 系统服务运行于 Session 0，与用户会话隔离
- 权限维持：Payload 脱离用户登录，依附高权限服务长期驻留
- 权限提升：目标服务进程若具备 SYSTEM 或高完整性权限，Payload 可执行高权限操作
- 隐蔽性：Session 0 不直接与用户交互，常规用户态监控难以检测
